# HARDWARE DEL ROBOT-VEHÍCULO PARA WRO
## PRIMERA VERSIÓN
Diseño electrónico y módulos empleados en el robot tipo coche a emplear en el concurso tipo WRO de la UCLM.
Listado de componentes Hardware:
1. ESPCAM.
2. Servo continuo.
3. Servo de posición.
4. Batería y cargador LIPO 18650.
5. Cableado.

Listado de materiales:
1. Tablero contrachapado de madera de 30x30x3cm.
2. 4 x Tornillos de M3.5x10.
3. 4 x Tornillos de M3.5x25.
4. 4 x muelles de D4x10.
5. Resina para impresora 3D, para engranajes.
6. 1 x varilla roscada de M4 y 30cm de longitud.

## VERSIÓN DEFINITIVA
El diseño se ha hecho utilizando los materiales que había en el taller de Tecnología del instituto.
Listado de componentes Hardware:
1. HuskyLens v2.
2. Tarjeta Micro:bit junto con su shield de expansión.
3. Sensor de medida de ultrasonidos HC-SR04
4. Portapilas de 6 pilas tipo AA.
5. 1 servo continuo
6. Ruedas x 5.
7. Un servo de posición para la dirección del vehículo.
8. Bastidor y elementos de la dirección.
9. Bridas de plástico de 5mm de ancho y 2mm de grosor x 10.
